About

If you're looking for pointers (no pun intended) on how to use git, look no further! The goal with this site is to make it usable as a way for beginners to get off the ground using Git and to show those who are already familiar with Git some new tricks.

Want to change the layout, design, or something else? Visit the project page on GitHub. Want your tip featured? Submit it here.

Want to get a hold of the author? Email him at nick@quaran.to, poke him on Twitter, or message him on GitHub.

A disclaimer about the tips: The author does not claim to be flawless or otherwise perfect in his git abilities. The main reason this blog was started was to learn more about Git and how it works. This is a learning process for everyone involved, and that's why it's valuable for both readers and the author. If you see mistakes or inaccuracies please don't hesitate to contact Nick or leave a comment.

And finally...thanks for reading!
